## Setup
After cloning the repository
```bash 
cd word-search-puzzle-solver
cd bin
1. Select main.exe
2. Input the name of the file you want to solve (ex: `test.txt`)
3. If you want to solve another puzzle, type `y`, otherwise, type `n`
```
